# C Programming Core Practices & Coursework

This repository contains fundamental exercises, algorithms, memory management practices (pointers and structs), and course assignments developed using the C programming language during my first year as a Computer Engineering student.

## 📁 Repository Structure & Contents

* **`math_and_logic_problems.c`**
  * Palindrome numbers, digit logic algorithms, grading system calculations, and basic geometric area computations using `switch-case`.
* **`number_theory_and_loops.c`**
  * Number representation systems (Octal/Hexadecimal), character code mapping (ASCII), loop-based number theory (perfect numbers, abundant numbers, divisor calculations), and sequence tracking.
* **`pointers_arrays_and_functions.c`**
  * Applied memory management using pointers (`pass-by-reference`), array manipulation, coin breakdown calculation, digit analysis, and custom array rotation algorithms.
* **`structs_enums_and_pointers.c`**
  * Custom data types using `struct` and `enum`, array of structs (book inventory system), and user data passing via values vs pointers.

## 🛠️ Built With
* **Language:** C (C99 standard)
* **Compiler:** GCC / Clang / MSVC
